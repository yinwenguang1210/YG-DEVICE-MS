# 关于

YG-DEVICE-MS 是一款**美观**、**流畅**、**功能丰富**、**高效**的串口调试工具，支持以下特点：

- 同时打开多个串口进行监听
- 串口日志自动保存
- 可以设置波特率、位大小等串口设置
- 可发送各种 AT 指令
- 具有各种特性
